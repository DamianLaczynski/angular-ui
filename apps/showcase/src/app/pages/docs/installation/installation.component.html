Installation

Step-by-step guide to install Angular UI in your project.

Prerequisites

Before installing Angular UI, make sure you have:

  • Node.js (version 18 or higher)
  • Angular CLI (version 17 or higher)
  • An existing Angular project

Installation via NPM

Install Angular UI using npm:

npm install angular-ui

Or using yarn:

yarn add angular-ui

Import Styles

Add the Angular UI styles to your angular.json file:

"styles": [
  "node_modules/angular-ui/styles/main.scss",
  "src/styles.scss"
]
