Getting Started

Learn how to install and use Angular UI components in your project.

Installation

Install Angular UI using npm:

npm install angular-ui

Basic Usage

Import the components you need in your Angular module or standalone component:

import { ButtonComponent } from 'angular-ui';

@Component({
  selector: 'app-example',
  imports: [ButtonComponent],
  template: `
    <ui-button variant="primary">Click me</ui-button>
  `
})
export class ExampleComponent {}

Theming

Angular UI supports both light and dark themes out of the box.

You can customize the theme by overriding CSS variables in your global styles.
